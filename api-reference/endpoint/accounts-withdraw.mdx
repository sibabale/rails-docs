---
title: "Withdraw Funds"
api: "POST https://api.railsinfra.com/api/v1/accounts/{id}/withdraw"
description: "Withdraw funds from an account"
---

Withdraws funds from a specific account.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Request

<ParamField body="amount" type="number" required>
  Amount to withdraw (must be greater than 0 and not exceed account balance)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Response

<ResponseField name="account" type="object">
  Updated account object with new balance
</ResponseField>

<ResponseField name="transaction" type="object">
  Transaction record for the withdrawal
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const result = await rails.accounts.withdraw("123e4567-e89b-12d3-a456-426614174000", {
  amount: "100.00",
  description: "ATM withdrawal"
});

console.log("New balance:", result.account.balance);
console.log("Transaction ID:", result.transaction.id);
```

## Error Codes

| Code | Description |
|------|-------------|
| `validation_error` | Amount must be greater than zero |
| `insufficient_funds` | Account balance is less than withdrawal amount |
| `account_not_found` | Account ID does not exist |
| `account_suspended` | Account is suspended and cannot process withdrawals |
| `account_closed` | Account is closed and cannot process withdrawals |
| `unauthorized` | Invalid or missing API key |
