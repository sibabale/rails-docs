---
title: "Create User"
api: "POST https://api.rails.com/api/v1/users"
description: "Create a new user (customer or employee). Public SDK endpoint."
---

Creates a new user in the Rails banking system. This is the only Users endpoint in the public API. Send your API key and environment in **headers**; the request body contains only user fields.

## Headers (required inputs)

<ParamField header="X-API-Key" type="string" required>
  Your API key. Required for all requests.
</ParamField>

<ParamField header="X-Environment" type="string" required default="sandbox">
  `sandbox` or `production`. Defaults to **sandbox** when not provided. Always use sandbox for development and testing.
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json` for request body.
</ParamField>

## Body

<ParamField body="first_name" type="string" required>
  User's first name
</ParamField>

<ParamField body="last_name" type="string" required>
  User's last name
</ParamField>

<ParamField body="email" type="string" required>
  Valid email address (must be unique)
</ParamField>

<ParamField body="password" type="string" required>
  User's password (sent securely; use HTTPS only)
</ParamField>

## Response

<ResponseField name="user_id" type="string">
  Unique identifier for the created user (UUID)
</ResponseField>

<ResponseField name="status" type="string">
  User status (e.g. active)
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const user = await rails.users.create({
  firstName: "Jane",
  lastName: "Doe",
  email: "jane@example.com",
  password: "secure-password",
  environment: "sandbox"
});

console.log("User created:", user.userId);
```

```go Go
package main

import (
  "context"
  "log"
  "os"

  "github.com/rails/rails-go"
)

func main() {
  client := rails.NewClient(rails.WithAPIKey(os.Getenv("RAILS_API_KEY")))
  user, err := client.Users.Create(context.Background(), rails.UserCreateParams{
    FirstName:   "Jane",
    LastName:    "Doe",
    Email:       "jane@example.com",
    Password:    "secure-password",
    Environment: "sandbox",
  })
  if err != nil {
    log.Fatal(err)
  }
  log.Println("User created:", user.UserID)
}
```

## Notes

- **Headers are required:** `X-API-Key` and `X-Environment` must be sent as headers, not in the body. `X-Environment` defaults to **sandbox** when omitted (SDK or client may send it explicitly).
- GET user, list users, and update user are not part of the public SDK; they are internal or admin-only.
