---
title: "Create Account"
api: "POST https://api.rails.com/v1/accounts"
description: "Create a new bank account for a user"
---

Creates a new bank account for an existing user. The account number is automatically generated.

## Request

<ParamField body="account_type" type="string" required>
  Type of account: `checking` or `saving`
</ParamField>

<ParamField body="user_id" type="string" required>
  UUID of the account owner
</ParamField>

<ParamField body="currency" type="string">
  Currency code (defaults to USD)
</ParamField>

<ParamField body="organization_id" type="string">
  Organization ID (optional)
</ParamField>

<ParamField body="environment" type="string">
  Environment: `production` or `sandbox` (defaults to production)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the account
</ResponseField>

<ResponseField name="account_number" type="string">
  Auto-generated account number
</ResponseField>

<ResponseField name="account_type" type="string">
  Type of account: `checking` or `saving`
</ResponseField>

<ResponseField name="user_id" type="string">
  ID of the account owner
</ResponseField>

<ResponseField name="balance" type="number">
  Initial account balance (0.00)
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code
</ResponseField>

<ResponseField name="status" type="string">
  Account status (always `active` for new accounts)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of account creation
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const account = await rails.accounts.create({
  accountType: "checking",
  userId: "456e7890-e89b-12d3-a456-426614174000",
  currency: "USD"
});

console.log("Account created:", account.account_number);
console.log("Account ID:", account.id);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    account_type: "checking",
    user_id: "456e7890-e89b-12d3-a456-426614174000",
    currency: "USD"
  })
});

const account = await response.json();
```

</RequestExample>

<ResponseExample>


# Deposit Funds

<api>POST https://api.rails.com/v1/accounts/{account_id}/deposit</api>

Deposits funds into a specific account.

## Path Parameters

<ParamField path="account_id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Request

<ParamField body="amount" type="number" required>
  Amount to deposit (must be greater than 0)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Response

<ResponseField name="account" type="object">
  Updated account object with new balance
</ResponseField>

<ResponseField name="transaction" type="object">
  Transaction record for the deposit
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
const result = await rails.accounts.deposit("123e4567-e89b-12d3-a456-426614174000", {
  amount: 500.00,
  description: "Salary deposit"
});

console.log("New balance:", result.account.balance);
console.log("Transaction ID:", result.transaction.id);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts/123e4567-e89b-12d3-a456-426614174000/deposit', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 500.00,
    description: "Salary deposit"
  })
});

const result = await response.json();
```

</RequestExample>

<ResponseExample>

```json Success Response
{
  "account": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "account_number": "ACC001234567",
    "balance": 500.00,

  },
  "transaction": {
    "id": "abc12345-e89b-12d3-a456-426614174000",
    "transaction_type": "deposit",
    "amount": 500.00,
    "description": "Salary deposit",
    "status": "completed",
    "created_at": "2024-01-15T15:20:00Z"
  }
}
```

</ResponseExample>

# Withdraw Funds

<api>POST https://api.rails.com/v1/accounts/{account_id}/withdraw</api>

Withdraws funds from a specific account.

## Path Parameters

<ParamField path="account_id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Request

<ParamField body="amount" type="number" required>
  Amount to withdraw (must be greater than 0 and not exceed balance)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Example

<RequestExample>

```typescript TypeScript SDK
const result = await rails.accounts.withdraw("123e4567-e89b-12d3-a456-426614174000", {
  amount: 50.00,
  description: "ATM withdrawal"
});

console.log("New balance:", result.account.balance);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts/123e4567-e89b-12d3-a456-426614174000/withdraw', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 50.00,
    description: "ATM withdrawal"
  })
});

const result = await response.json();
```

</RequestExample>

# Transfer Funds

<api>POST https://api.rails.com/v1/accounts/{from_account_id}/transfer</api>

Transfers funds from one account to another.

## Path Parameters

<ParamField path="from_account_id" type="string" required>
  Source account ID (UUID format)
</ParamField>

## Request

<ParamField body="to_account_id" type="string" required>
  Destination account ID (UUID format)
</ParamField>

<ParamField body="amount" type="number" required>
  Amount to transfer (must be greater than 0 and not exceed source balance)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Response

<ResponseField name="from_account" type="object">
  Updated source account with new balance
</ResponseField>

<ResponseField name="to_account" type="object">
  Updated destination account with new balance
</ResponseField>

<ResponseField name="transaction" type="object">
  Transaction record for the transfer
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
const result = await rails.accounts.transfer("123e4567-e89b-12d3-a456-426614174000", {
  toAccountId: "789e1234-e89b-12d3-a456-426614174000",
  amount: 100.00,
  description: "Transfer to savings"
});

console.log("Transfer completed:", result.transaction.id);
console.log("From balance:", result.from_account.balance);
console.log("To balance:", result.to_account.balance);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts/123e4567-e89b-12d3-a456-426614174000/transfer', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to_account_id: "789e1234-e89b-12d3-a456-426614174000",
    amount: 100.00,
    description: "Transfer to savings"
  })
});

const result = await response.json();
```

</RequestExample>

<ResponseExample>


## Error Codes

| Code | Description |
|------|-------------|
| `insufficient_funds` | Account balance is less than withdrawal/transfer amount |
| `invalid_amount` | Amount must be greater than zero |
| `account_not_found` | Source or destination account does not exist |
| `account_suspended` | Account is suspended and cannot perform transactions |
| `account_closed` | Account is closed and cannot perform transactions |
| `same_account_transfer` | Cannot transfer to the same account |
