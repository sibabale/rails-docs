---
title: "Get Account Transactions"
api: "GET https://api.railsinfra.com/api/v1/accounts/{account_id}/transactions"
description: "Retrieve transaction history for a specific account"
---

Retrieves the transaction history for a specific account.

## Path Parameters

<ParamField path="account_id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Query Parameters

<ParamField query="limit" type="number">
  Maximum number of transactions to return (default: 50)
</ParamField>

## Response

Returns an array of transaction objects.

<ResponseField name="id" type="string">
  Unique transaction identifier (UUID)
</ResponseField>

<ResponseField name="account_id" type="string">
  Account ID associated with the transaction
</ResponseField>

<ResponseField name="transaction_type" type="string">
  Type of transaction: `deposit`, `withdrawal`, or `transfer`
</ResponseField>

<ResponseField name="amount" type="number">
  Transaction amount (decimal)
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (e.g., `USD`)
</ResponseField>

<ResponseField name="description" type="string">
  Transaction description (if provided)
</ResponseField>

<ResponseField name="status" type="string">
  Transaction status: `completed`, `pending`, or `failed`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of transaction creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const transactions = await rails.transactions.listByAccount("123e4567-e89b-12d3-a456-426614174000", {
  limit: 25
});

console.log("Transactions:", transactions.length);
```

```go Go
transactions, err := client.Transactions.ListByAccount(ctx, "123e4567-e89b-12d3-a456-426614174000", rails.TransactionListByAccountParams{
  Limit: int64(25),
})
```

## Error Codes

| Code | Description |
|------|-------------|
| `account_not_found` | Account ID does not exist |
| `unauthorized` | Invalid or missing API key |
| `invalid_uuid` | Invalid UUID format for account ID |
