---
title: "Get Account"
api: "GET https://api.rails.com/v1/accounts/{account_id}"
description: "Retrieve account details and balance information"
---

Retrieves detailed information for a specific bank account including balance and status.

## Path Parameters

<ParamField path="account_id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the account
</ResponseField>

<ResponseField name="account_number" type="string">
  Account number (auto-generated)
</ResponseField>

<ResponseField name="account_type" type="string">
  Type of account: `checking` or `saving`
</ResponseField>

<ResponseField name="user_id" type="string">
  ID of the account owner
</ResponseField>

<ResponseField name="balance" type="number">
  Current account balance (decimal)
</ResponseField>

<ResponseField name="currency" type="string">
  Three-letter currency code (e.g., `USD`)
</ResponseField>

<ResponseField name="status" type="string">
  Account status: `active`, `suspended`, or `closed`
</ResponseField>

<ResponseField name="organization_id" type="string">
  Organization ID (if applicable)
</ResponseField>

<ResponseField name="environment" type="string">
  Environment: `production` or `sandbox`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of account creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const account = await rails.accounts.get("123e4567-e89b-12d3-a456-426614174000");

console.log("Account Type:", account.account_type);
console.log("Balance:", account.balance);
console.log("Status:", account.status);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts/123e4567-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const account = await response.json();
```

</RequestExample>

# List User Accounts

<api>GET https://api.rails.com/v1/accounts</api>

Retrieves all accounts for a specific user.

## Query Parameters

<ParamField query="user_id" type="string" required>
  User ID to filter accounts (UUID format)
</ParamField>

## Response

Returns an array of account objects with the same structure as the single account endpoint.

## Example

<RequestExample>

```typescript TypeScript SDK
const accounts = await rails.accounts.list({
  userId: "456e7890-e89b-12d3-a456-426614174000"
});

console.log("User has", accounts.length, "accounts");
accounts.forEach(account => {
  console.log(`${account.account_type}: $${account.balance}`);
});
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts?user_id=456e7890-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer your_api_key_here',

});

const accounts = await response.json();
```

</RequestExample>

# Get Account Transactions

<api>GET https://api.rails.com/v1/accounts/{account_id}/transactions</api>

Retrieves transaction history for a specific account.

## Path Parameters

<ParamField path="account_id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Query Parameters

<ParamField query="limit" type="number">
  Maximum number of transactions to return (default: 50)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique transaction identifier
</ResponseField>

<ResponseField name="transaction_type" type="string">
  Type of transaction: `deposit`, `withdrawal`, `transfer`
</ResponseField>

<ResponseField name="amount" type="number">
  Transaction amount (decimal)
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code
</ResponseField>

<ResponseField name="description" type="string">
  Transaction description
</ResponseField>

<ResponseField name="status" type="string">
  Transaction status: `completed`, `pending`, `failed`
</ResponseField>

<ResponseField name="created_at" type="string">
  Transaction timestamp
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
const transactions = await rails.accounts.getTransactions("123e4567-e89b-12d3-a456-426614174000", {
  limit: 10
});

console.log("Recent transactions:", transactions.length);
transactions.forEach(txn => {
  console.log(`${txn.transaction_type}: $${txn.amount} - ${txn.description}`);
});
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/accounts/123e4567-e89b-12d3-a456-426614174000/transactions?limit=10', {
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const transactions = await response.json();
```

</RequestExample>

<ResponseExample>

## Error Codes

| Code | Description |
|------|-------------|
| `account_not_found` | Account ID does not exist |
| `access_denied` | Insufficient permissions to view this account |
| `validation_error` | Invalid query parameters |
